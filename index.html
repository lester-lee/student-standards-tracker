        Help

        Selection

        There are dropdown menus at the top of the page that allow the viewer to choose individual students or courses.

        Student View (top)

        On the left side of the student view is a heatmap that gives an overview of a single student's mastery of each standard. Viewers can tell at a glance which standards a student has met, and which standards a student should work on.

        The right side provides another view of the same data, sorted by mastery. This allows for a more focused understanding of which standards the student has not yet met.

        Course View (bottom)

        On the bottom is aggregrate information about the selected course. It shows the ratio of student mastery for each standard, and at a glance provides viewers with an understanding of which standards the class as a whole has met or not yet met.

        About This Project

        Hi! I'm Lester Lee, and this is my final project for Interactive Data Visualization taught by Ellie Frymire in Fall 2021.

        This is a proof-of-concept project for an app that can be used by educators to keep track of student progress in their courses, as well as to provide accessible and visual feedback to students.

        More information about this project and my process can be found on the porfolio page.

        Data Sources

        The data used for this visualization is publically accessible on Github.

        This data is currently randomized to protect the privacy of my students; while no "real" analysis can be done on the publically available data, this is a tool that I use to analyze the grades of my current students.
